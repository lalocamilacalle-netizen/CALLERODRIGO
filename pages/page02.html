HARDWARE

Parte Física del computador

Componentes Internos

CPU

El procesador es el cerebro del computador. Ejecuta instrucciones, procesa datos y coordina todas las operaciones del sistema. Es fundamental para el rendimiento general del equipo.

Memoria RAM

La memoria RAM almacena datos de manera temporal mientras el computador está en uso. Permite ejecutar múltiples programas de forma rápida y eficiente.

Disco Duro

Dispositivo de almacenamiento permanente que guarda el sistema operativo, programas y archivos del usuario.

Placa Madre

Es la base del computador donde se conectan todos los componentes internos. Permite la comunicación entre ellos.

Tarjeta de Video

Procesa gráficos e imágenes, esencial para videojuegos, diseño gráfico y edición multimedia.

Fuente de Poder

Convierte la corriente eléctrica y distribuye energía a todos los componentes del computador.

Periféricos de Entrada

Teclado

Permite ingresar datos y comandos al sistema mediante teclas.

Mouse

Dispositivo apuntador que permite interactuar con la interfaz gráfica del sistema.

Cámara Web

Captura imágenes y video en tiempo real para videollamadas o grabaciones.

Periféricos de Salida

Monitor

Muestra la información visual procesada por el computador.

Impresora

Permite obtener documentos físicos impresos en papel.

Altavoces

Reproducen sonido generado por el sistema.

Otros Dispositivos

Router

Permite la conexión a internet y comunica diferentes redes.

Switch

Conecta múltiples dispositivos dentro de una red local.

Módem

Convierte señales para permitir la conexión a internet.
